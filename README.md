# STM32_FreeRTOS_Queue_Processing

## This application implements the below commands

1-LED_ON

2-LED_OFF

3-LED_TOGGLE_START

4-LED_TOGGLE_STOP

5-LED_STATUS_READ

6-RTC_DATETIME_READ


### The command should be sent to board via UART from the user
